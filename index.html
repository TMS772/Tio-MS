TM walks through life as a quiet learner — one who listens more than he speaks,
who finds meaning not in answers, but in the gentle rhythm of asking.
For him, learning is a slow art of becoming;
a way to touch truth without claiming to own it.
And so he lives by a simple light:
“Every Day Is My Happy Day.”

Website Optimization Checklist

  • 1️⃣ File & Struktur
  • [ ] Pastikan index.html berada di root repository
  • [ ] Semua file CSS/JS/Assets sudah terupload dan path benar
  • [ ] Folder assets rapi, sesuai struktur
  • 2️⃣ Gambar & Media
  • [ ] Gunakan format ringan (WebP/JPEG)
  • [ ] Ukuran gambar tidak terlalu besar (<500 KB)
  • [ ] Resolusi gambar sesuai kontainer / layout
  • 3️⃣ Font & Tipografi
  • [ ] Hanya load font yang dipakai (Playfair Display 400 & 700)
  • [ ] Ukuran font sesuai untuk desktop & mobile
  • [ ] Periksa line-height & warna teks (#7a674d)
  • 4️⃣ Responsif / Mobile-Friendly
  • [ ] Meta viewport sudah benar
  • [ ] Cek tampilan di HP, tablet, desktop
  • [ ] Media queries aktif untuk font & padding
  • 5️⃣ Performance
  • [ ] Minify CSS / HTML jika perlu
  • [ ] Tidak ada script / CSS yang tidak dipakai
  • [ ] Loading halaman < 2 detik
  • 6️⃣ Navigasi & UX
  • [ ] Hamburger menu berfungsi di semua device
  • [ ] Section kosong siap dikembangkan
  • [ ] Link internal / anchor bekerja lancar
  • 7️⃣ Testing & Maintenance
  • [ ] Cek website di Google PageSpeed
  • [ ] Pastikan tidak ada error di console browser
  • [ ] Commit terbaru sudah di-push ke GitHub
  • [ ] Backup repository jika ada perubahan besar
